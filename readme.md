# vim
 把vim(neovim)配置成日常开发(golang)使用的的ide
 
##  文件介绍
* .tmux.conf   
   tmux是一个好用的终端复用器类似iterm，但是要比iterm好用，此配置与vim没有任何关系（我的目的是留一个备份）
* coc-settings.json  
   coc.nvim的核心配置文件 
* neovim.config   
   我自己的neovim的配置文件，仅供参考，可自行修改。
